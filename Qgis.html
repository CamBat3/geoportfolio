Bicycle accident Hotspots
Description

The aim for this project was to use open-source software to create a product from spatial data. Qgis an open-source spatial tool was used for all the analysis, and all data was obtained from open-source sites. The analysis was focused on bike accidents within Melbourne CBD during Morning peak-hour and Evening peak-hour.

As you can see from the below maps, there is a clear difference between where the hotspots occur during the morning and evening. The maps were created using CARTO which allowed for each level of risk to be easily identifiable. They were then put into a website format and hosted online, unfortunately the website is no longer up.
